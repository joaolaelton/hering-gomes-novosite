# Project: Hering Gomes (Site Institucional de Advocacia)

## Visão Geral
Sistema/Site voltado para o escritório de advocacia Hering Gomes. Focado em transparecer profissionalismo e autoridade, integrando um backend em Supabase para conteúdo dinâmico (depoimentos, blog, advogados, etc.).

## Stack Tecnológica
- **Frontend:** React / Vite
- **Styling:** TailwindCSS
- **Backend / Database:** Supabase

## Papéis (Squads Envolvidos)
- **@frontend-design:** UI/UX premium, landing page de alta conversão, design system robusto.
- **@gsd-nativa:** Gerenciador de estado, checklists de tarefas e milestones.
- **@supabase-mcp (Backend/DB):** Queries e integrações banco de dados (Depoimentos, etc).
